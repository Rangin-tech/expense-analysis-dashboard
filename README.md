# Expense Analysis Dashboard

## 📊 Overview

This project analyzes expense data using Microsoft Excel to identify spending patterns and trends.

## 🛠 Tools Used

* Microsoft Excel (Pivot Tables, Charts)

## 📈 Features

* Category-wise expense analysis
* Data visualization using charts
* Dashboard for insights

## 🔍 Insights

* Highest spending categories: Bills, Food, Shopping, Transport
* Lowest spending category: Entertainment
* Spending is evenly distributed across major categories
